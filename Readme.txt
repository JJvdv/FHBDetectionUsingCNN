AgriCultureCNN.ipynb - Python notebook for all code to train the CNN model.
ResNet18.ipynb - Python notebook for call code to train the ResNet18 model.
ModelApi.py - API for inference of the two models with FastAPI library .
CNN.py - Python file to train the CNN model.
NeuralNetwork.py - NeuralNetwork class - A Convolutional Neural Network with all the layers.
requirements.txt - All required libraries to be installed to run the application.
WebAppImageClass - The Django project for the front-end for model inference. Given a .pt file, a prediction and image will be returned.
